JavaScript

Javascript es un lenguaje de programacion o de secuencias de comendos que nos permite poder implementar funciones complejas en nuestras paginas web, cosas que con solo html y css no podriamos realizar como por ejemplo: mostrar contenido que se actualice de forma dinamica, mapas interactivos, graficos animados en 2D/3D, etc.

JavaScript fue creado en 1995 por Brendan Eich mientras trabajaba en Netscape Communications, el lenguaje fue desarrollado originalmente bajo el nombre de Mocha y despues LiveScript para despues llamarse definitivamente como JavaScript el nombre que todos conocemos hoy en dia.

La primera version de JavaScript se integro en el navegador Netscape Navigator 2.0, el cual fue lanzado en diciembre de 1995

JavaScript vs Java

JavaScript y Java aunque compartan el nombre de "Java" fue por solo mercadotecnia ya que Netscape se aprovecho de la popularidad de Java en ese momento para poder atraer a mas desarrolladores, por eso empezo siendo Mocha y despues LiveScript para al final quitar el Live por Java, pero en si son diferentes uno de otro ya que Java es un lenguaje de programacion de proposito general orientada a objetos, que requiere compilacion y se ejecuta en una maquina virtual Java (JVM), permitiendo crear aplicaciones de escritorio, servidores y aplicaciones moviles en Android. En cambio JavaScript es un lenguaje de programacion interpretado que se ejecuta en el navegador del cliente, permitiendo crear aplicaciones web interactivas y dinamicas.

JavaScript con Html y css

JavaScript se usa con HTML y css para poder darle mas dinamismo a nuestra pagina, sabemos que html es la estructura y que css son los estilos pero javascript se encarga de la logica y de la interactividad, juntos son los complementos para poder crear una pagina web.

Fundamentos del lenguaje

En JavaScript la sintaxis es muy parecida a otros lenguajes de programación como java o c++, pero tiene diferencias cruciales como por ejemplo que en javascript para definir una variable no se especifica el tipo de dato como en otros lenguajes que hay que especificar si es de tipo entero, cadena o flotante, etc. En javascript solo basta con inicializarla para poder usar el tipo de dato que querramos.

Diferencias entre var, let y const

En javascript tu puedes declarar variables de 3 formas diferentes, con: var, let y const

  • var: Es la forma antigua de declarar variables y estas pueden ser globales o de funcion y se pueden redeclarar
  • let: Esta forma de declarar variables se diferencia porque es por bloques osea se usa en ciclos y no se puede redeclarar en el mismo bloque
  • const: Las variables declaradas con const no pueden ser actualizadas ni redeclaradas. Deben ser inicializadas en el momento de la declaracion y su valor no puede cambiar.

Tipos de dato primitivos

Un tipo de dato primitivo se refiere a un valor basico que no es un objeto y no tiene metodos ni propiedades asociadas, en javascript los tipos de dato primitivo son:

  • Number: para representar numeros enteros y de punto flotante
  • String: para representar cadenas de texto
  • Boolean: para representar valores true o false
  • Null: para representar un vacio intencional
  • Undefined: para representar una variable que ha sido declarada pero no inicializada
  • Symbol: para representar un valor unico e inmutable
  • BigInt: para representar numeros enteros grandes que exceden el limite de Number

Arrays y objetos

En JavaScript, los arrays y los objetos son dos de las estructuras de datos más importantes.

Arrays

Un array es una lista ordenada de valores. Los valores pueden ser de cualquier tipo, incluyendo otros arrays y objetos. Los arrays se definen utilizando corchetes [] y los elementos se separan por comas.

Objetos

Un objeto es una colección de pares clave-valor. Las claves son cadenas y los valores pueden ser de cualquier tipo. Los objetos se definen utilizando llaves {} y los pares clave-valor se separan por comas, los objetos cuentan con propiedades y metodos osea acciones que este realiza

Operadores logicos, aritmeticos y relacionales

Los operadores se dividen en 3 tipos, los aritmeticos, logicos y relacionales, estos se usan para diversas operaciones, comparaciones y asignaciones.

  • Operadores aritmeticos:Este tipo de operadores son los que normalmente usamos en matematicas y que vemos seguido, sirven para poder realizar operaciones basicas con numeros. Los operadores aritmeticos son: + (suma), - (resta), * (multiplicacion), / (division), % (modulo)
  • Operadores logicos:Este tipo de operadores se utilizan para realizar operaciones logicas y booleanas. Los operadores logicos son: && (AND), || (OR), ! (NOT)
  • Operadores relacionales:Este tipo de operadores se utilizan para comparar valores. Los operadores relacionales son: == (igualdad), === (igualdad estricta), != (desigualdad), !== (desigualdad estricta), < (menor que),> (mayor que), <= (menor o igual que),>= (mayor o igual que)

Control de Flujo

Sentencias condicioneales

Las sentencias condicionales son estructuras que permiten tomar decisiones en el flujo de un programa. En JavaScript, las sentencias condicionales más comunes son:

  • if: Permite ejecutar un bloque de código si se cumple una condición.
  • else: Permite ejecutar un bloque de código si la condición del if no se cumple.
  • else if: Permite comprobar múltiples condiciones.
  • switch: Permite ejecutar diferentes bloques de código según el valor de una expresión.

Bucles

Los bucles son estructuras que permiten repetir un bloque de código varias veces. En js los bucles que se usan son:

  • for: Permite repetir un bloque de código para determinado número de veces.
  • while: Permite repetir un bloque de código mientras se cumpla una condición.
  • do while: Permite repetir un bloque de código hasta que se cumpla una condición.

Break y continue

Las sentencias break y continue se utilizan para controlar el flujo de los bucles.

  • break: Permite salir de un bucle de forma anticipada.
  • continue: Permite saltar a la siguiente iteración de un bucle.

Funciones

Una funcion es un bloque de codigo que se puede reutilizar, las funciones pueden recibir parametros y pueden devolver un valor, las funciones en js se pueden definir de varias formas:

  • Declaracion de funcion: Se define con la palabra reservada function seguida del nombre de la funcion y los parametros entre parentesis.
  • Expresion de funcion: Se define asignando una funcion anonima a una variable.
  • Funciones flecha: Se define utilizando la sintaxis de flecha (=>) y son una forma mas concisa de escribir funciones.

Parametros y argumentos

Los parametros son las variables que se definen en la declaracion de la funcion y los argumentos son los valores que se pasan a la funcion cuando se llama.

Valor de retorno

El valor de retorno es el valor que devuelve una funcion cuando se llama, se utiliza la palabra reservada return para especificar el valor que se quiere devolver.

Propiedades y metodos diferencias

Las propiedades son las caracteristicas o atributos de un objeto, mientras que los metodos son las acciones o funciones que un objeto puede realizar. Por ejemplo en el objeto persona, las propiedades son nombre, edad y profesion, mientras que el metodo es saludar.

Metodos basicos de los arrays

  • push: Se usa para agregar un elemento al final del array.
  • pop: Este elimina el ultimo elemento del array.
  • map: Nos sirve para crear un nuevo array con los resultados de aplicar una funcion a cada elemento del array original.
  • shift: Elimina el primer elemento del array.
  • unshift: Agrega un elemento al principio del array.
  • forEach: Ejecuta una funcion en cada elemento del array.
  • filter: Crea un nuevo array con los elementos que cumplen una condicion.

Otros temas basicos

Que es DOM

El DOM (Document Object Model) es una interfaz de programación que permite a los scripts acceder y manipular el contenido, la estructura y el estilo de un documento HTML o XML. El DOM representa el documento como un árbol de nodos, donde cada nodo corresponde a un elemento o un atributo del documento.

Eventos en JavaScript

Un evento es una señal de que algo ha sucedido en una pagina web como las interacciones del usuario como cuando da clicks, pulsasiones de tecla o eventos del navegador como cuando carga la pagina

  • Eventos del raton: click (un clic) ,mouseover (el puntero cuando pasa por un elemento), mouseover (cuando el cursor se mueve)
  • Eventos del teclado: keydown (se presiona una tecla), keyup (se suelta una tecla)
  • Eventos de la pagina: load (se termina de cargar la pagina), resize (cambia el tamaño de la ventana)

Alert, prompt y console.log

Estas 3 funciones sirven para imprimir un mensaje pero cada una sirve para cosas diversas desde la forma en que lo imprime y como lo muestra:

  • alert: Este sirve para mostrar un mensaje emergente en la pagina
  • prompt: Pide un dato al usuario como el alert pero se usa para solicitar texto
  • console.log: Este sirve para mostrar un mensaje en la consola del navegador
